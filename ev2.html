ライブ EV 計算（スマホ完結）

重い学習は GitHub Actions（スマホの GitHub から起動）が実施し、各馬の 独立勝率 π（斤量・馬体重・騎手・過去走から算出、オッズ非依存）を ev2_upcoming.json に書き出します。このページは発走前に ライブ単勝オッズだけを取得し、p ∝ πα·qβ、 EV = p × オッズ を計算します。オッズは閲覧時に取得するので常に最新です。

π データを読み込み、レースを選んで「ライブオッズ取得・計算」を押してください。

バックテスト予測の閲覧

Python 側 evmodel.pipeline が walk-forward 検証で書き出した ev2_predictions.json（過去レースの結合勝率 p・市場勝率 q・勝敗）を読み込み、 EV>1 の買い目・実績 ROI・Kelly 配分を表示します。理論は docs/EV_METHODOLOGY.md。
